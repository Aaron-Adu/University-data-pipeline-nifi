University Data Warehouse Project

Apache NiFi + PostgreSQL + Prometheus/Grafana

About the Project

This project simulates a production-grade, on-premise data warehouse setup using Apache NiFi. It includes real data flows such as CSV file ingestion, public API data transformation, Apache log parsing, and database-to-database migration.

Features

  • Multiple ETL/ELT pipelines using Apache NiFi
  • Integration with PostgreSQL and Prometheus/Grafana
  • Handling CSV, JSON (API), Logs, and SQL data sources
  • Error routing, timestamping, schema inference
